# CPPageControl
分页控件pageControl

```
CPPageControl *page = [[CPPageControl alloc] initWithFrame:CGRectMake(0, 100, self.view.frame.size.width, 20)];
    page.numberOfPages = 4;
    page.currentPage = 0;
    [self.view addSubview:page];
```
